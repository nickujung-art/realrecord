# PRD: 리얼레코드 창부이 데이터랩

## 목표

창원·김해 아파트 실거래 신고가를 실시간으로 추적하고, 거래취소를 감지하여 지역 부동산 실수요자·투자자에게 데이터 기반의 투명한 정보를 제공한다.

> "오늘 우리 동네 챔피언은? 창원·김해 부동산의 심장박동을 읽다"

---

## 타겟 사용자

| 세그먼트 | 니즈 |
|---|---|
| **실거주자** | 내가 살 아파트 갭가격, 학군, 관리비, 주차 환경 한눈에 확인 |
| **투자자** | 신고가 경신 알림, 거래취소 감지, 매물 수 추이 (수급 파악) |
| **공인중개사** | 프리미엄 광고 노출로 잠재 고객 유입 (수익 모델) |

---

## 핵심 기능

### 완료된 기능 (Sprint 1–8)

1. **실거래 신고가 대시보드**
   - 최신 신고가 거래 목록 (오늘의 왕 카드 포함)
   - 날짜 기준: 가장 최근 거래일 기준 동적 계산 (하드코딩 없음)

2. **아파트 상세 페이지** (`/apartments/[id]?pyeong=N`)
   - 평형별 역대 최고가 배지
   - 가격 히스토리 차트 (SmartChart: 기간 필터 + AI 아웃라이어 토글)
   - 최근 거래 20건 목록
   - 갭가격 (매매 - 전세) — 실 전세 데이터 미연동 시 null 표시
   - 매물 수 추이 (saleCount/rentCount, 전월 대비 %)
   - 학군 정보 (NEIS 학교알리미, 도보 시간 + 등급)
   - 관리비 이력 12개월 (계절별 비교 바)
   - 시설 정보 (주차, 승강기, 난방방식, 복도유형)
   - 포털 직접 링크 (네이버 부동산, KB부동산)
   - 거래 취소 경고 배지

3. **단지 검색**
   - 실시간 검색 (debounce 250ms)
   - 한글 초성 검색 지원 (예: "ㅊㅇ" → 창원)

4. **광고주 수익 모델**
   - 아파트-광고주 N:M 연결
   - 상세 페이지 내 프리미엄 중개사 카드 노출
   - 어드민 UI (`/admin/advertisers`)

5. **공공데이터 수집 파이프라인**
   - MOLIT 실거래 (약 7만 건, 창원 + 김해)
   - KB부동산 매물 수 (puppeteer 배치)
   - K-APT 시설 정보 + 관리비
   - NEIS 학교알리미

## MVP 제외 사항

- 알림 (push / email / 카카오 알림톡) — 향후 로드맵
- 모바일 앱 — 웹 반응형으로 우선 대응
- 타 지역 확장 (창원·김해 외) — 데이터 검증 후 단계적 추가
- 커뮤니티 기능 (댓글, 좋아요) — 핵심 데이터 완성 후 검토
- 실시간 WebSocket 알림 — 배치 수집 우선

## 디자인 방향

- **대시보드 도구**: 마케팅 랜딩이 아닌, 매일 쓰는 업무 도구처럼 설계
- **다크 테마 고정**: 데이터 밀도가 높아 눈 피로 최소화
- **데이터 밀도 우선**: 스크롤 최소화, 한 화면에 많은 정보
- **AI 슬롭 금지**: glass morphism, gradient-text, 네온 글로우, 보라 브랜드 색상 전부 금지 (UI_GUIDE.md 참조)
